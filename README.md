# 科研课题项目管理系统 java991

## 项目概述

科研课题项目管理系统是一款针对科研项目管理需求设计的软件，旨在规范项目管理流程，提高管理效率。

## 技术栈

- 后端：SpringBoot
- 前端：Vue
- 数据库：MySQL
- 缓存：Redis

## 运行环境

- JDK 1.8
- MySQL 8.0.X
- Node.js 12.X.X

## 功能模块

- 用户管理
- 项目管理
- 申报管理
- 变更管理
- 结题管理
- 角色管理
- 权限管理
- 数据字典

## 系统角色

系统支持多角色登录管理，具体角色根据项目需求定制。

## 目录结构

```
- 后端代码目录
  - src
    - main
      - java
        - com.example
          - controller
          - service
          - repository
          - entity
  - resources
- 前端代码目录
  - src
    - api
    - components
    - views
    - store
    - router
```

## 系统特点

- 前后端分离架构，前后端完全独立
- 支持多角色登录管理
- 项目全生命周期管理
- 权限控制灵活
- 数据字典维护方便

## 核心亮点

- 提供科研项目管理必备工具
- 支持多维度数据分析
- 规范项目管理流程

## 部署步骤

1. 准备运行环境，安装JDK 1.8、MySQL 8.0.X和Node.js 12.X.X
2. 部署后端代码，构建并运行SpringBoot项目
3. 部署前端代码，构建并运行Vue项目
4. 初始化数据库，导入系统所需表结构和数据
5. 启动系统，开始使用

注意：在部署过程中，请确保环境配置正确，避免出现版本兼容性问题。
